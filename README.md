## Getting Started

This project require NodeJS v12.
Recommended NodeJS v14.

### Install
Clone this project

```bash
> git clone https://github.com/lyfla/basic-baileys.git
> cd basic-baileys
```

Install :

```bash
> npm install 
```

### Usage
Run the Whatsapp baileys

```bash
> node main
```

after running it you need to scan the qr
